# Rashmitha349-UNP_PROJECT_IRRIGATION_WATER_PREDICTION_GEN_AI_PYTHON


UNP_PROJECT_IRRIGATION_WATER_PREDICTION_GEN_AI_&_PYTHON
📌 Overview
This project predicts irrigation water requirements using environmental factors such as temperature, rainfall, and humidity. It is designed as a simple decision-support system for efficient water usage.

🔧 Approach

Used crop-related dataset (2200 × 8)
Created target variable (irrigation_mm) using domain logic
Performed EDA to understand feature relationships
Trained regression models for prediction
📊 Results

Model	MAE	RMSE	R² Score
Linear Regression	1.15	1.47	0.965
Random Forest	1.32	1.66	0.956
Gradient Boosting	1.36	1.71	0.954
Linear Regression performed best, indicating largely linear relationships.

💻 Deployment
A simple interface using Gradio allows users to input values and get irrigation predictions.
